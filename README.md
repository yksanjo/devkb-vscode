# DevKB VS Code - Developer Knowledge Base (VS Code Extension)

A VS Code extension for knowledge management directly in your IDE.

## Features

- **Ask** - Ask questions about your codebase from VS Code
- **Search** - Search knowledge entries
- **Add** - Add knowledge entries from selected code
- **Index** - Index current project
- **Stats** - View statistics

## Installation

1. Open VS Code
2. Search for "DevKB" in Extensions
3. Install and configure API URL in settings

## Configuration

```json
{
  "devkb.apiUrl": "http://localhost:3001",
  "devkb.dataDir": ".devkb"
}
```
